# Calculate the percentage of a number

Write a JavaScript function to calculate the percentage (%) of a number.

```
Test Data: 
console.log(percentage(1000, 47.12));
471.2 
```

**JavaScript Code:**

```js
function percentage(num, per)
{
  return (num/100)*per;
}
          
console.log(percentage(1000, 47.12));

```
