# JavaScript: Empty an array keeping the original

Write a JavaScript script to empty an array keeping the original.

::: tip

**Sample Solution:**

:::

**HTML Code:**

```html




  
  JavaScript script to empty an array keeping the original.






```

**JavaScript Code:**

```javascript
arr = [1, 3, 6, 3, -5];
console.log(arr);
arr.length = 0;
console.log(arr);

```
